# Industry Research Skill

面向 VC 的行业与赛道深度研究工具。自主搜集权威信源，生成基于事实的行业研究报告。

## 核心特性

- 🔍 **自主调研**：自动搜索 6 大维度（行业概览、产业链、竞争格局、技术、政策、趋势）
- 📊 **数据溯源**：每个数据点标注来源 URL 和可信度等级
- 🚫 **零编造**：严格基于公开信息，无法验证的内容明确标注或省略
- 📄 **双格式输出**：Markdown + Word，方便编辑和分享
- 🎯 **VC 视角**：专为判断赛道和投资机会设计

## 快速开始

```bash
# 安装依赖
pip install -r requirements.txt

# 运行研究（以 AI 漫剧为例）
python industry_research.py "AI漫剧" --deep

# 输出文件
# ./output/AI漫剧_行业研究报告_2026-04-27.md
# ./output/AI漫剧_行业研究报告_2026-04-27.docx
```

## 六大研究维度

| 维度 | 核心问题 |
|------|----------|
| 行业概览与市场规模 | 行业是什么？有多大？增长多快？ |
| 产业链结构与价值链 | 上下游是谁？利润怎么分？壁垒在哪？ |
| 主要公司与竞争格局 | 谁在玩？份额多少？模式差异？ |
| 技术发展与演进路径 | 核心技术是什么？走到哪了？下一步？ |
| 政策环境与监管动态 | 政府支持还是限制？合规成本高吗？ |
| 未来趋势与投资启示 | 方向去哪？风险在哪？值不值得看？ |

## 可信度分级

- 🟢 **高**：McKinsey、Gartner、IDC、政府统计、上市公司财报
- 🟡 **中**：36氪、TechCrunch、行业垂直媒体
- 🔴 **低**：自媒体、未经验证来源

## 使用场景

1. **看新项目前**：快速扫描赛道，判断 TAM 和竞争格局
2. **投资委员会**：准备行业背景材料，支撑投资决策
3. **Portfolio 管理**：监控已投赛道动态，识别风险与机会
4. **年度复盘**：系统性梳理关注赛道的发展变化
