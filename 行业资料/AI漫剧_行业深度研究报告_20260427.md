## 执行摘要（Executive Summary）

**研究对象**：AI漫剧（AI动画微短剧），指借助AIGC技术实现自动化或半自动化生产的动画微短剧内容，单集时长通常1-5分钟，以网文/漫画IP改编或原创剧本为素材，通过AI视频生成模型完成视觉内容生产。

### 核心发现

1. **市场规模爆发式增长**：2025年中国动画微短剧市场规模达**189.8亿元**，同比激增**276.3%**（艾媒咨询）。预计2030年将突破**850亿元**。用户规模从2025年的约1.2亿快速增至2026年的**2.8亿**（DataEye研究院）。

2. **平台与资本全面入场**：字节（红果短剧/Seedance）、腾讯（阅文/火龙漫剧）、百度（柚漫剧/七猫漫剧）、快手（可灵AI）等互联网巨头于2025年下半年至2026年初密集入局。2026年春节档，AI漫剧首次跻身各平台春节档，标志大厂全面入局。

3. **技术驱动成本重构**：Seedance 2.0将视频可用率从约20%提升至**90%**，单项目制作成本降幅达**80%**（中银证券测算）。AI漫剧单分钟制作成本从万元量级降至**千元量级**，单部成本可压缩至十几万元，制作周期从3个月压缩至**10-20天**。

4. **监管强规范落地**：2026年4月1日起，广电总局将AI漫剧正式纳入网络微短剧分类分层审核体系，实行"先备案后上线"。三级审核制度（投资≥300万总局审核 / 100-300万省级审核 / <100万平台自审报备）大幅提高准入门槛。

5. **出海成为第二增长曲线**：海外市场有望成为新蓝海。谷歌调研显示，在美国、巴西、印尼三大市场，**55%**的观众愿意为AIGC微短剧付费。昆仑万维旗下DramaWave海外收入达**54.1亿元**，占公司总收入的93%。

6. **行业二八效应极端**：华福证券统计显示，2025年累计播放量不足百万的漫剧达30,042部，占比约**80%**；而突破5亿播放量的仅1部。尾部作品无人问津，头部效应极其显著。

7. **头部VC叙事**：a16z 将 2025 定义为"视频之年"，2026 为"Agentic 视频编辑之年"；Runway 设立 $10M 基金投资 AI 视频生态早期公司，验证"视频智能"（Video Intelligence）作为下一代多模态AI入口的叙事。

## 一、行业概览与市场规模

### 1.1 行业定义与边界

AI漫剧（AI动画微短剧）是**漫剧的细分类型**，属于微短剧赛道的一个垂直分支。根据艾媒咨询2025年12月发布的《2025-2026年中国AI漫剧行业趋势白皮书》定义：

> "AI漫剧是借助AIGC技术实现自动化生产的动画微短剧。"

其核心特征包括：
- **内容形态**：基于网文、漫画、表情包等图文素材，通过AI视频生成技术转化为动态视频内容
- **时长特征**：单集1-5分钟，每部通常75-100集，总时长约90-150分钟
- **生产模式**："创意输入 → AI生成 → 人工优化"的高效链路，AI参与度约70%-80%
- **视觉风格**：涵盖2D动态漫、3D漫剧、AI仿真人漫剧等多种形态

**行业边界说明**：AI漫剧与传统动画番剧、真人微短剧存在交叉但边界清晰——
- vs 传统番剧：单集更短、成本更低（降幅超90%）、制作周期以天计算而非月/年
- vs 真人微短剧：省去演员、场地、设备等成本，不存在真人演员的档期与片酬约束

### 1.2 历史发展阶段

AI漫剧行业在2024-2026年间经历了从萌芽到爆发的快速演进：

| 时间 | 里程碑事件 | 意义 |
|------|-----------|------|
| 2024年末 | 真人微短剧爆发后，资本与流量焦点向AIGC转移 | 行业萌芽 |
| 2025年初 | 酱油、灵矩、花生等团队率先入局，验证商业模式 | 商业验证期 |
| 2025年5月 | 灵矩动漫启动AI漫剧项目，月产从2-3部快速扩至50-70部 | 产能爬坡 |
| 2025年10月 | 各大平台纷纷推出官方漫剧合作计划 | 平台加持 |
| 2025年11月 | 红果短剧率先推出独立漫剧App；广电总局启动专项治理 | 政策介入 |
| 2025年12月 | 百度推出柚漫剧App；艾媒咨询发布行业白皮书 | 行业元年确认 |
| 2026年1月 | 百度七猫推出"七猫漫剧"App；腾讯上线"火龙漫剧"App | 大厂全面入场 |
| 2026年2月 | AI漫剧首次跻身各平台春节档；央视频官宣AI漫剧大会 | 主流化 |
| 2026年4月1日 | 广电总局备案新规强制执行，未备案作品全网下架 | 强监管时代 |

> **行业共识**：2025年被普遍定义为"AI漫剧元年"，2026年有望迈入工业化与精品化双轮驱动的全新发展阶段（证券时报，2026-02-05）。

### 1.3 市场规模与增长

#### 1.3.1 整体市场规模

根据艾媒咨询数据：
- **2025年中国动画微短剧市场规模达189.8亿元**，同比激增276.3%
- 另有行业口径（搜狐/新浪财经）显示"国内AI漫剧市场规模突破168亿元"，与艾媒咨询数据量级一致
- **预计2030年将突破850亿元**（艾媒咨询预测）

**增速驱动因素**：
1. **技术突破**：Seedance 2.0、可灵3.0等模型将视频可用率从20%提升至90%，直接降低80%制作成本
2. **平台入场**：字节、腾讯、百度等大厂以流量+资金+IP三重优势快速做大市场
3. **用户习惯养成**：Z世代（3亿核心人群）对二次元内容的强付费意愿
4. **出海增量**：海外市场成为新增长蓝海

#### 1.3.2 用户规模

| 指标 | 数值 | 来源 |
|------|------|------|
| 2025年漫剧用户规模 | 约1.2亿 | DataEye研究院 |
| 2026年预计用户规模 | 约2.8亿 | DataEye研究院 |
| 核心用户年龄分布 | 18-23岁（23.07%）、24-30岁（24.04%）、31-40岁（31.81%） | DataEye研究院 |

#### 1.3.3 播放量与供给数据

- **抖音2025年全年AI漫剧累计播放量达757.7亿**，下半年播放量复合增长率高达24%，12月单月创超200亿播放量纪录（搜狐，2026-02）
- **2025年12月单月，抖音端原生上线37,583部**漫剧（新浪财经，2026-02-05）
- **2025年全年，我国累计注册8.02万家漫剧相关企业**，同比增37.1%；华东地区注册量最多，占比32.2%（腾讯新闻/企查查，2026-02-05）
- 2025年全国共有3.3万部微短剧上线播出，AI漫剧占比持续提升（新浪财经，2026-02-26）

#### 1.3.4 与真人微短剧的对比

| 维度 | AI漫剧 | 真人微短剧 |
|------|--------|-----------|
| 制作周期 | 10-20天 | 2-3个月 |
| 团队规模 | 3-7人 | 30+人 |
| 单部成本 | 10-50万元 | 30-150万元 |
| 毛利率 | >70%（头部口径） | 约30-50% |
| 产能瓶颈 | 算力/模型能力 | 演员/场地/档期 |
| 内容风险 | 版权/肖像/魔改 | 演员塌房/片酬 |
| 用户付费率 | 超80%愿付费（部分口径） | 约20-30% |

> **重要说明**：关于"毛利率超70%"和"超八成用户愿意付费"的数据来自创业邦（2026-01-16）等媒体报道，但缺乏第三方独立验证，置信度为**中低**。实际行业调研显示，约90%的AI漫剧公司处于亏损状态（虎嗅，2026-03-09）。

### 2.2 上游分析：IP与技术

#### 2.2.1 IP供给方

IP是AI漫剧产业链的**核心上游资源**。当前行业IP来源呈现以下特征：

- **网文平台占绝对主导**：番茄小说（字节）、七猫（百度）、阅文集团（腾讯）等拥有海量网文IP库
- **平台开放IP授权**：2025年下半年起，阅文、七猫、番茄小说开放**数万部IP授权**（虎嗅，2026-03-09）
- **阅文集团**：以"IP+AI"为核心，开放10万部精品IP、设立亿元创作基金，推出AIGC工具"漫剧助手"
- **百度七猫**：开放10万部IP资源，与酱油文化达成深度合作
- **原创剧本占比提升**：部分头部团队开始采用AI生成原创剧本，以规避IP授权成本

#### 2.2.2 技术工具方

AI大模型与AIGC工具是行业降本增效的**技术底座**：

| 技术环节 | 代表工具/平台 | 厂商 |
|----------|-------------|------|
| 视频生成 | Seedance 2.0、即梦 | 字节跳动 |
| 视频生成 | 可灵AI | 快手 |
| 视频生成 | 通义万相 | 阿里巴巴 |
| 视频生成 | 混元大模型 | 腾讯 |
| 全流程工具 | 万兴剧厂 | 万兴科技 |
| 全流程工具 | 巨日禄 | 第三方 |
| 剧本生成 | 漫剧助手 | 阅文集团 |
| 配音配乐 | 各类TTS/音乐AI | 多家 |

**关键技术数据**：
- 2025年中国AI大模型市场规模约**495.4亿元**，预计2026年突破**700亿元**（新浪财经，2026-02-26）
- Seedance 2.0生成15秒视频的可用率提升至**90%**（凤凰网/极客公园，2026-02-07）
- 生成一条15秒视频的成本约为**15元**（API约1元/秒），C端高级会员约**0.2元/秒**（中国企业家，2026-04-01）

### 2.3 中游分析：内容制作

中游是AI漫剧产业链的**核心生产环节**。当前呈现**"大厂自建+工作室+个人创作者"**三层结构：

#### 2.3.1 大厂自建团队

- **字节-星漫**：依托抖音流量+番茄IP+Seedance技术，形成完整闭环
- **百度-柚漫剧/七猫漫剧**：2025年12月推出柚漫剧App，2026年1月推出七猫漫剧App，双线布局
- **快手-漫创**：依托可灵AI技术，推出创投计划

#### 2.3.2 中型专业团队

- **酱油动漫**（阅文投资）：2025年11月抖音端漫剧新增播放量TOP30中，酱油文化（阅文关联）占据重要份额
- **灵矩动漫**：2025年5月启动项目，年底月产达50-70部，预计2026年3月可达150部/月
- **与光同尘**：从3人团队扩张到300人，靠AI漫剧实现千万营收
- **灵境万维**：落地武汉半年发行超300部AI漫剧，爆款《我在末世开超市》上线5天播放量3亿、收入1200万元，制作成本仅15万元（武汉科技局，2026-03-24）

### 2.4 下游分析：分发与变现

#### 2.4.1 分发渠道格局

- **抖音/红果短剧**：绝对头部，依托5亿+用户流量池，18-35岁用户占比72%
- **快手**：去中心化分发，评论活跃度较强
- **B站**：聚焦年轻二次元群体，侧重精品化、IP向漫剧
- **腾讯（火龙漫剧/QQ/微信）**：2026年2月上线火龙漫剧App，依托阅文IP生态
- **百度（柚漫剧/七猫漫剧）**：2025年12月-2026年1月密集推出
- **美团**：2026年初在主App内上线"饭团漫社"，成为新的大厂玩家（21世纪经济报道，2026-02-27）

#### 2.4.2 变现模式

AI漫剧的变现模式与真人短剧高度相似，但呈现更轻量化的特征：

| 变现模式 | 说明 | 收入占比 | 特点 |
|---------|------|---------|------|
| **IAA（广告变现）** | 贴片广告、中插广告、激励视频（看30秒广告解锁一集） | 约70% | 主要收入来源，长尾收入 |
| **IAP（应用内购买）** | 前几集免费，关键剧情付费解锁 | 约30% | 直接转化核心用户，快速回款 |
| **平台分账** | 按播放量、完播率、付费金额等指标分级分成 | 变动 | 核心收益分配机制 |
| **IP衍生** | 手办、游戏、真人长剧改编、潮玩等 | 初期 | 长尾价值，《Dragon King》手办销量破10万件 |
| **出海收入** | 海外平台付费/广告 | 快速增长 | DramaWave海外收入54.1亿元 |

*来源：金元证券研究所（2025-12）、huiyelaw.com（2026-01-04）*

#### 2.4.3 投放与流量成本

- **巨量千川**作为核心投放渠道，占比达到**66%**（新浪财经，2026-02-26）
- 字节在漫剧赛道的单日广告消耗已冲上**2000万元**（创业邦，2026-01-16）
- 快手同期日消耗约**700万元**
- 海外市场投流成本："15万美元制作+150万美元投流"成为回本铁律（华夏时报，2026-02-10）

### 2.5 价值链利润分配

AI漫剧产业链的价值分配呈现**"平台拿大头、制作方赚辛苦钱"**的特征：

```
用户付费/广告收入
      │
      ▼
┌─────────────┐
│   平台方     │ ← 抽取30-50%分成
│ （抖音/快手等）│
└─────────────┘
      │
      ▼
┌─────────────┐
│   制作方     │ ← 获得剩余50-70%，扣除成本后净利有限
│ （工作室/MCN）│
└─────────────┘
      │
      ▼
┌─────────────┐
│   IP方/工具方 │ ← 授权费或工具使用费
│              │
└─────────────┘
```

> **关键洞察**：据虎嗅（2026-03-09）报道，行业内有估算称**90%的AI漫剧公司处于亏损状态**。核心原因在于：
> 1. 流量成本（投流）侵蚀大部分收入
> 2. 尾部作品无人问津，ROI无法打正
> 3. 平台政策变动（如红果收紧保底）导致承制公司批量倒闭

### 3.2 字节跳动 vs 腾讯：核心对决

当前AI漫剧赛道最核心的一对竞争关系是**字节跳动 vs 腾讯阅文**（创业邦，2026-01-16；投资界，2026-02-23）：

| 维度 | 字节跳动 | 腾讯（阅文） |
|------|---------|------------|

| **IP来源** | 番茄小说（海量免费网文） | 阅文集团（10万+精品IP，含超级IP） |
| **流量优势** | 抖音+红果5亿+用户，算法精准分发 | 微信+QQ+视频号生态，IP粉丝基础深厚 |
| **技术工具** | Seedance 2.0（业内公认最强视频生成模型） | 混元大模型 |
| **变现效率** | 投流能力强，单日消耗2000万+ | 投流能力弱于字节，依赖IP口碑驱动 |
| **内容策略** | 80%依赖外部采购/UGC，内容供给不稳定 | 以自有IP改编为主，精品化路线 |
| **出海布局** | Melolo（东南亚），"免费+金币提现"模式 | 相对滞后 |
| **核心软肋** | 内容生态可持续性、IP深度不足 | 流量运营经验不足、漫剧触达范围有限 |

### 3.4 行业二八效应

华福证券以超3万部漫剧为统计样本分析指出（证券时报，2026-02-05）：

## 四、技术发展与演进路径

### 4.1 核心技术栈

AI漫剧的制作流程通常分为四个阶段（huiyelaw.com，2026-01-04）：

1. **剧本创作**：AI辅助剧本改编与分镜生成（人工干预为主）
2. **视觉生成**：AI生成角色、场景、分镜画面
3. **动画化与视频合成**：视频生成模型完成动态合成
4. **配音配乐**：AI配音（TTS）+ AI配乐

**全流程AI参与度**：约70%-80%，但"故事构思、剧本分镜写作、剪辑目前往往仍主要人工完成"。

### 4.2 关键模型与工具对比

#### 4.2.1 Seedance 2.0（字节跳动）

- **定位**：目前公认的"地表最强"视频生成模型之一（极客公园，2026-02-07）
- **核心突破**：多模态输入极致受控，最多可同时输入**12个参考文件**约束视频生成
- **可用率**：从约20%跃升至**90%**
- **成本**：生成一条15秒视频约**15元**（API）；C端高级会员约0.2元/秒
- **瓶颈**：算力紧张，高峰期排队3-7小时；2026年4月连续三次涨价
- **API策略**：仅部分开放，白名单主要面向大型影视公司，最低消费可达**1000万元/年**

#### 4.2.2 可灵AI（快手）

- **定位**：B端专业用户优先，画面细节更丰富、电影感强
- **成本**：每秒视频生成成本约**0.07美元**，已实现推理层面毛利率转正
- **商业化**：2025年收入1.4亿元，2026年1月月活1200万，ARR超3亿美元
- **瓶颈**：排队时间超30分钟

#### 4.2.3 其他模型

| 模型 | 厂商 | 特点 |
|------|------|------|
| 通义万相 | 阿里巴巴 | 阿里系视频生成 |
| 混元大模型 | 腾讯 | 腾讯生态嵌入 |
| Vidu Q3 | 生数科技 | 全球首个支持16秒音画直出 |
| Sora 2.0 | OpenAI | 海外最强，但成本极高（年化烧钱54亿美元） |

### 4.3 成本重构效应

Seedance 2.0带来的成本重构是行业爆发的**关键技术拐点**：

> 中银证券测算：以制作一部90分钟漫剧项目为例——
> - 原有20%可用率水平下：实际需生成素材约450分钟，成本约**1万余元**
> - 可用率提升至90%后：所需生成素材压缩至100分钟，成本降至约**2000元**
> - **单项目成本降幅达80%**

华泰证券研判：随着新一代模型规模化应用，AI漫剧单集制作周期有望从7-10天压缩至**3天以内**。

### 4.4 技术瓶颈

尽管技术进步显著，AI漫剧制作仍面临以下技术挑战（中国企业家，2026-04-01；36氪，2026-03-09）：

1. **角色一致性**：同一角色在不同镜头中可能出现面部/服装/场景不一致
2. **物理合理性**："六根手指""360度旋转手臂"等物理破绽仍频繁出现
3. **"抽卡"困境**：创作者80%时间在反复生成中筛选可用素材
4. **算力瓶颈**：高峰排队8小时+，算力成本上升倒逼平台涨价
5. **版权溯源**：AI生成内容的版权归属尚无法律明确定论

### 4.5 技术演进方向

基于当前技术路线和厂商动态，未来1-2年的技术演进可能包括：

1. **"一键成片"Agent**：从分镜到成片的全流程自动化，进一步压缩人工参与
2. **多模态一致性模型**：解决角色/场景一致性问题
3. **算力成本下降**：随着模型优化和算力扩容，生成成本有望继续降低
4. **版权存证技术**：区块链等技术用于AI生成内容的权属存证

## 五、政策环境与监管动态

### 5.1 监管框架演变

AI漫剧行业在2025-2026年经历了从"平台自审"到"广电+平台双重把关"的监管升级：

| 时间 | 政策事件 | 核心内容 |
|------|---------|---------|
| 2025年9月 | 广电总局《管理提示（动画微短剧管理）》 | 首次明确将动画微短剧纳入微短剧管理范畴 |
| 2025年11月 | 广电总局专项治理 | 首次将AIGC类、漫画类、表情包类动画微短剧纳入分类分层审核体系 |
| 2026年1月 | "AI魔改"专项治理 | 清理违规视频超5.2万条、处置违规账号140余个 |
| 2026年2月 | 广电总局《关于进一步统筹发展和安全促进网络微短剧行业健康繁荣发展的通知》 | 从许可准入、备案管理、规划引导、内容审核、行业自律等方面统筹强化 |
| 2026年4月1日 | 备案新规强制执行 | 未备案存量作品全网强制下架，新作"先备案后上线" |

*来源：国家广播电视总局官网、杭州日报、21世纪经济报道等*

### 5.2 三级分层审核制度

2026年4月1日起执行的备案新规核心内容（21世纪经济报道，2026-04-10；杭州日报，2026-04-09）：

| 投资规模 | 审核层级 | 审核主体 | 许可证类型 |
|---------|---------|---------|-----------|
| ≥300万元，或涉敏感题材 | 一级 | 国家广电总局 | 《网络剧片发行许可证》 |
| 100万-300万元，普通题材 | 二级 | 省级广电部门 | 备案号 |
| <100万元，一般题材 | 三级 | 平台自审后报省级备案 | 备案登记 |

### 5.3 内容红线

新规划定的明确内容红线包括：

1. **严禁低俗擦边**：危害青少年身心健康的台词与画面
2. **严禁AI魔改经典**：颠覆性篡改四大名著、革命题材、英模人物等
3. **严禁未经授权使用真人肖像**：AI换脸/盗用肖像构成侵权
4. **强制AI标识**：AI生成内容必须在片头或显著位置标注"AI生成"
5. **未成年人保护**：适龄作品标注提示、平台落实儿童标签专人审核

> **实际影响**：新规落地后，全网日上新量从150部暴跌至**20部**（163.com，2026-03-31）。低质侵权作品被批量清理，行业从"谁都能做"转向"谁做得好"。

### 5.4 版权与合规风险

AI漫剧行业面临的版权与合规问题日益突出：

- **侵权案件高发**：知乎盐言故事跟进的短剧、漫剧侵权案件达**68起**，超40起已完成取证（证券时报，2026-02-05）
- **明星肖像被盗用**：杨紫、肖战、迪丽热巴、易烊千玺等均遭遇AI换脸侵权。2026年4月5日，易烊千玺工作室发布声明要求下架侵权作品
- **红果短剧处置**：完成对1.5万部作品的核查，处置违规作品**670部**（杭州日报，2026-04-09）
- **法律空白**：AI生成内容的版权归属及侵权认定标准尚不完善，亟需加快立法（解放日报，2026-04-23）

### 5.5 地方扶持政策

与此同时，地方政府也在积极扶持AI漫剧产业：

- **北京**：2025年11月发布《北京市促进"人工智能+视听"产业高质量发展行动方案(2025-2029)》，鼓励动漫、微短剧行业的AI技术应用
- **南宁**：出台专项政策，企业落户最高奖400万；拍摄制作补贴每集5000元，单部最高10万
- **郑州**：2025年9月召开微短剧高质量发展大会，推出系列行业扶持举措
- **武汉**：灵境万维等企业落地，地方科技局积极扶持

## 六、头部 VC Insight 与投资叙事

### 6.1 a16z：Agentic Video 与多模态智能

a16z 在 2026 年 1 月发布博客《It's time for agentic video editing》（a16z.com，2026-01-21），明确提出：

> "2025 was the year of video. 2026 is the year we let agents edit it."

**核心论点**：
1. **视频已赢**：视频是人类学习、营销和连接的主要方式，但编辑瓶颈持续增长
2. **Agent 编辑三阶段**：
   - **自动化（Automate）**：AI 完成字幕、粗剪、多平台适配等重复任务
   - **适配（Adapt）**：将长视频自动切割为适合不同平台的短片段，包括翻译和重新配音
   - **优化（Optimize）**：Agent 具备"品味（taste）"，能够理解叙事节奏、音乐情绪和观众钩子
3. **技术成熟度已到位**：视觉模型、工具使用 Agent 和海量训练数据在过去一年均已成熟

**对 AI 漫剧的映射**：a16z 的叙事虽然面向广义视频编辑，但其"Agentic 视频"框架直接适用于 AI 漫剧的工业化生产——从剧本到分镜到成片的全流程 Agent 化，正是当前漫剧行业技术演进的方向。

### 6.2 Runway：$10M 基金与"视频智能"生态

Runway 于 2026 年 3 月宣布设立 **$10M 基金** 和 Builders 计划（TechCrunch，2026-03-31），投资基于其 AI 视频模型的早期公司。

**投资论点的三个方向**：
1. 推动 AI 前沿的技术团队（新架构）
2. 在基础模型之上构建应用层的团队（新用例）
3. 试验新媒体创作、叙事和分发的公司（新媒介）

**已披露的被投公司**：
- **LanceDB**：AI 应用数据库基础设施
- **Tamarind Bio**：AI 设计新蛋白质的生物公司
- **Cartesia**：实时音频生成公司

**叙事信号**：Runway 从"创意工具"向"通用世界模型"（General World Models）转型，试图将视频生成能力扩展至更广泛的行业应用。对于 AI 漫剧而言，这意味着上游视频生成模型的能力边界正在快速外推。

### 6.3 中国 VC 视角：技术+内容+出海

根据云沐资本、正轩投资、柏睿资本等中国 VC 的公开表态：

- **云沐资本**（Elser.AI 天使轮跟投）："坚定看好团队在产品创新与场景落地上的持续突破，期待加速成长与全球化拓展"
- **柏睿资本**（灵境AI 天使轮）："灵境AI 跳出单纯的工具思维，致力于将人工智能深度融入业务流程，展现出 AI 时代优秀企业的卓越远见"
- **零以创投**（灵境AI 天使轮）："灵境通过 AI 技术可大幅提升动漫生产效率和降低生产成本，构建了'数据飞轮+AI技术'的双壁垒"

**中国 VC 的共识叙事**：
1. AI 漫剧是 AIGC **率先跑通商业模型**的垂类赛道之一
2. 技术壁垒不在于单点工具，而在于**全流程 Pipeline + 数据飞轮**
3. **出海**是确定性增量，海外动漫市场规模数千亿元

## 七、早期创业公司扫描（A轮及以前）

### 7.1 Elser.AI

- **成立时间**：约2024年
- **总部**：中国（具体城市未公开）
- **核心产品**：AI漫剧创作Agent + 海外漫剧APP
- **商业模式**：ToB（短剧/漫剧AI Agent工具）+ ToC（海外漫剧内容平台）
- **融资历史**：
  - 种子轮：奇绩创坛投资（2025年上半年）
  - 天使轮：正轩投资领投，后浪资本、云沐资本跟投，累计数百万美元（2025年11月）
- **团队背景**：创始人北大哲学博士背景，团队聚焦AI漫剧Agent研发。从文生漫画工具起步，在零投放情况下收获20万+注册用户。
- **差异化**：国内少有的将"AI Agent"概念深度应用于漫剧全流程生产的公司，强调低门槛、高质量、高效率的创作体验。
- **亮点**：产品已内测完毕，Agent 2.0版即将上线；团队兼具学术背景和工程能力
- **风险**：早期阶段，商业化路径尚需验证；面临大厂（字节/百度/腾讯）同类工具的直接竞争

### 7.2 灵境AI（灵境万维）

- **成立时间**：2023年（杭州），2025年换赛道至AI影视
- **总部**：杭州 → 武汉（获地方科技局扶持）
- **核心产品**：AI动漫工业化生产平台，基于DIT架构的AIGC全链路制作工具
- **商业模式**：自制内容 + 代工服务 + 出海平台
- **融资历史**：
  - 天使轮：数千万元人民币，柏睿资本、零以创投投资，高鹄资本担任FA
- **团队背景**：创始人许金城，提出"神经渲染+认知计算"双核驱动理念。团队自研AIGC制作工具，打通小说改编、剧本分镜、静图生成、动态渲染全链路。
- **差异化**：将生产成本降至传统动漫的1/20，生产效率提升10倍；每分钟制作成本仅数百元人民币；强调"数据飞轮"效应。
- **亮点**：落地武汉半年发行超300部AI漫剧；爆款《我在末世开超市》上线5天播放量3亿、收入1200万元，制作成本仅15万元；计划月产500部+代工500部
- **风险**：快速扩张中的品控风险；DIT架构的技术护城河是否足够深

### 7.3 景鹰科技（Reel.AI）

- **成立时间**：2021年
- **总部**：中国（上海/北京）
- **核心产品**：全球首款AI短剧APP「Reel.AI」，商业短视频AI+人工协同创作SaaS平台
- **商业模式**：AI短剧内容平台 + SaaS工具
- **融资历史**：
  - 种子轮：真格基金千万级人民币（2021年11月）
  - 种子+轮：数百万人民币（2022年10月）
  - Pre-A轮：九合创投、众麟资本数百万美元（2023年11月）
  - Pre-A+轮：百度投资数百万美元（2024年）
- **团队背景**：创始人兼CEO朱江，上海交大计算机系，美国上市连续成功创业者，前触宝科技首席增长官，从零构建千万级DAU泛娱乐应用。核心团队来自百度、快手、滴滴、字节，毕业于上海交大、复旦、清华等高校。公司是国内最早基于大模型探索视频生成的团队之一。
- **差异化**：国内最早（2021年）基于大模型探索视频生成的团队之一；自研多模态大模型，AIGC视频生成质量达行业领先水平。
- **亮点**：获百度战略投资，验证技术路线；产品已上线海外市场并进入娱乐应用畅销榜单
- **风险**：公司成立较早但产品迭代周期较长；面临后发的大厂竞争压力

### 7.4 HOLYWATER

- **成立时间**：2020年
- **总部**：乌克兰（全球化运营）
- **核心产品**：MyDrama（垂直流媒体平台）、FreeBits（广告支持伴侣平台）、MyPassion（电子书）、MyMuse（AI驱动垂直系列）
- **商业模式**：多平台内容生态，AI辅助内容生产，广告+付费混合变现
- **融资历史**：
  - 战略投资：FOX Entertainment（2025年10月），承诺超200部系列内容合作
  - Series A：2200万美元（2026年1月），Horizon Capital领投，Endeavor Catalyst、Wheelhouse跟投
- **团队背景**：联合创始人 Bogdan Nesvit（牛津/UCL/联合国/BAT/Genesis 背景）和 Anatolii Kasianov（英国技术背景）。团队结合技术 expertise、全球视野和创业愿景。
- **差异化**：以乌克兰为基地但面向全球，强调"AI + 数据 + 全球经验"的移动叙事；与 FOX Entertainment 深度绑定IP资源。
- **亮点**：生态覆盖8500万+用户；获顶级娱乐集团 FOX 战略投资；Series A 规模在乌克兰/东欧创纪录

### 8.1 技术演进方向

**高置信度判断**（基于已确认的技术路线和厂商公开信息）：

2. **模型能力持续迭代**：Seedance、可灵等模型将持续升级，解决一致性、物理合理性等核心瓶颈
3. **算力成本中长期下降**：随着推理优化和算力扩容，生成成本有望继续降低，但短期受供需紧张影响可能维持高位

4. **多模态基座模型成为主流**：字节Seedance代表的多模态路线（文生图、文/图生视频统一模型）可能优于单视频模型路线
5. **端侧AI推理**：随着模型小型化，未来可能在端侧（手机/PC）完成部分AI生成任务

### 8.2 市场变化趋势

**高置信度判断**：

1. **两极分化加剧**：行业将呈现明确的"工业化量产 vs 精品化IP"两极分化（多家机构共识）
   - 一端：极致压缩成本，月产数百部的"内容工厂"
   - 另一端：聚焦中高成本精品，构筑长期IP价值
2. **平台流量向头部集中**：2026年平台将把**70%的流量资源倾斜给头部优质内容**（杭州日报，2026-04-09）
3. **出海成为确定性增量**：海外市场付费意愿验证（55%愿为AIGC付费），中国企业具备内容生产优势

**中置信度判断**：

4. **女频赛道崛起**：当前60%AI漫剧扎堆玄幻、末世题材，内容同质化严重。女频赛道（2026预计破200亿）是破局关键
5. **IP衍生价值释放**：漫剧作为"IP放大器"，成功作品将反向导流原著小说、漫画，并导向影视改编、游戏授权、衍生品开发

### 8.4 对投资人的启示

#### 值得关注的方向

1. **IP护城河玩家**：拥有海量IP储备的平台（阅文、中文在线、番茄/字节）具备先手优势。未来竞争本质是IP版权之战。
2. **出海基础设施**：AI漫剧出海需要本地化、投流、支付等基础设施。易点天下与阿里云的合作模式值得关注。
3. **精品化制作方**：在行业从"数量竞争"向"质量竞争"转型过程中，具备审美和叙事判断力的头部制作方（酱油动漫、灵矩动漫等）有望获得溢价。
4. **工具平台**：万兴科技等提供全流程AIGC工具的企业，受益于行业产能扩张。
5. **算力/云服务**：漫剧是token消耗的"吞金兽"，算力租赁平台和云服务商直接受益。
6. **早期创业公司**：Elser.AI（AI Agent方向）、灵境AI（工业化生产+出海）、景鹰科技（百度背书+自研模型）等具备差异化定位的早期公司。

#### 需要谨慎的方向

1. **纯代工/无IP的制作方**：平台政策变动风险高，毛利率被压缩，缺乏议价能力
2. **过度依赖单一平台的公司**：平台分成政策、审核标准变化将直接冲击业务
3. **题材单一的团队**：当前60%扎堆玄幻/末世，同质化竞争惨烈
4. **忽视合规的投机者**：2026年4月监管新规已明确，违规作品被批量下架

#### 关键观察指标

- **备案通过率**：反映监管松紧程度和内容质量底线
- **头部作品播放集中度**：反映行业二八效应是否缓和
- **出海收入占比**：反映第二曲线是否成型
- **模型可用率提升速度**：反映技术迭代的边际效果
- **平台分成政策变化**：直接影响产业链利润分配
